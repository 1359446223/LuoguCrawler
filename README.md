## 使用须知

自己在main.py同文件夹里创建cookie.json 然后填入你的cookie，如下。

``` json
{
    "cookie": "这里填你的luoguCookie"
}
```

其次，在main.py中修改

``` python
myUrl = "https://www.luogu.org/space/show?uid=72813"
myID = 72813
```
